What is currency?
Conversion rates designate how much of one currency is needed to purchase goods using another currency.

Those currencies shows that 1 USD in another country's currency like: euro, rubles, sums, yuan, franc, yen, and etc, would be *this* much. Let's look at some examples below where we will convert dollars to euro and dollars to rubles.

Converting $ to €

In this given code fragment, if user inputs 150 dollars, program will convert that number and output 126.88 in euro.
Here is how output will look like in the program: 150 American Dollars in European Euro will be 126.88
1 USD = 0.845867 EUR (Last checked on September 9 at 11pm)

Converting $ to ₽

Here is another example, let's convert dollars to rubles. For that, user inputs 100 dollars, and the program will output 7536 rubles.
Here is how output will look like in the program: 100 American Dollars in Russian Rubles will be 7536
1 USD = 75.3653 RUB (Last checked on September 9 at 11pm) 

//Code that converts dollars to euro
double dollars = 150; // input in dollars
double euro = dollars / 1.18222; // converting dollars to euro
System.out.println("dollars + "American Dollars in European Euro will be" + euro); // printing out the output in Euro

//Code that converts dollars to rubles
double dollars = 100; // input in dollars
double rubles = dollars * 75.36; // converting dollars to rubles
System.out.println("dollars + "American Dollars in Russian Rubles will be" + rubles); // printing out the output in Rubles
